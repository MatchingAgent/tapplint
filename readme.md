## Install

```bash
$ npm install tapplint
```

## Usage

```bash
Usage
  $ tapplint [<file|glob> ...]

Options
  --fix           Automagically fix issues
  --reporter      Reporter to use
  --stdin         Validate/fix code from stdin
  --ignore        Additional paths to ignore  [Can be set multiple times]
```
